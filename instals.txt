skrypt sh do odpalenia
```
#!/bin/bash

python3="/usr/bin/python3"

python_script="/home/pi/camera.py"

echo "start python camera serivce"
$python3 $python_script
```
uduchamiale: `chmod +x camera-srart.sh `

sudo vim /etc/systemd/system/camera.service
```
[Unit]
Description=Camera Python Script

[Service]
User=pi
Type=simple
ExecStart=/home/pi/camera-srart.sh
WorkingDirectory=/home/pi

Restart=always
[Install]
WantedBy=multi-user.target
```

reload services: `sudo systemctl daemon-reload`
run przy starupie: `sudo systemctl enable camera`
start: `sudo systemctl start camera`
logi:  `sudo journalctl -u camera`
